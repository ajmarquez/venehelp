Localizados Venezuela

Úsalo para buscar a una persona en los listados de pacientes de hospitales.

Plataforma pública y de código abierto que consolida listados de pacientes en hospitales para ayudar a ubicar a personas ya localizadas tras el terremoto. Incluye API pública de solo lectura.

Activa Voluntario Sitio web Personas localizadas

Reúne listados de pacientes en hospitales, no reportes de desaparecidos. Si buscas a alguien desaparecido, usa la sección Personas desaparecidas.
