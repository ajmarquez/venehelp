Localizados Venezuela

Use this to search for a person in hospital patient listings.

Open-source public platform that consolidates hospital patient listings to help locate people already found after the earthquake. Includes a public read-only API.

Active Volunteer Website Located people

It gathers hospital patient listings, not missing-person reports. To search for someone missing, use the Missing people section.

Notes

  • Public API documentation is available at /api and the v1 GET endpoints return JSON with CORS enabled for read access.
  • The project README explicitly says it is only for people already located and points missing-person cases to desaparecidosterremotovenezuela.com.
